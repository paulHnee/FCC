# JS5 - Weather App

A small weather checker that fetches current conditions for selected cities.

## Run

1. Open index.html in a browser.
2. Select a city from the dropdown.
3. Click "Get Weather" to load current conditions.

## Data source

- Uses the freeCodeCamp weather proxy API.

## Notes

- Requires an internet connection.
- Some fields may show N/A if the API omits data.
